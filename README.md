# Arcade2D

This monorepo contains individually packaged components to help with 2D game
development in JavaScript. This is mostly a personal project since I enjoy
programming these type of projects, but for that same reason the code quality is
kept to a high standard that makes these components usable in your own
production applications.

## Demos

This repository contains demos as a quickstart point. They are runnable with the
`yarn start:demo:*` command e.g.

```
$ yarn start:demo:td-shooter
```

Check the root `package.json` and/or the `demos/*` folders for others.
